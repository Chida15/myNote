### 四.用户管理

| ------------ | ------------ | ------------ | ------------------------------ | ---------------------- | ---------------- |
| /etc/passwd  | useradd      | /etc/group   | groups：当前账号支持的群组     | su                     | w                |
| /etc/shadow  | passwd       | /etc/gshadow | newgrp：有效群组(当前生效群组) | sudo                   | who              |
|              | usermod      |              | groupadd                       | visudo(设置sudo使用权) | lastlog          |
|              | userdel      |              | groupdel                       |                        | mail(用户间邮箱) |
|              |              |              | gpasswd(也可指定群管理员)      |                        |                  |
